# PermCalculator



### Usage: 

```
 Usage: ./a.out [OPTION] INPUT

 Options: 
		-s | --string	pass a string as an argument 
				e.g: ./a.out -s -rw-rw-r-- 
		-d | --digit	pass a per-integer 
				e.g: ./a.out -d 664
```

#### Example:

```
./a.out --digit 677
```

```
./a.out --string -rwxrwxrwx
```
